### 기능 테스트

#### 1. ✅ 미니 달력 열기
- **테스트**: 스케줄 페이지 상단 날짜 클릭
- **결과**: 미니 달력 + 예약 리스트 창이 함께 표시됨

#### 2. ✅ 예약 표시 (빨간 점)
- **테스트**: 캘린더에서 예약이 있는 날짜 확인
- **결과**: 날짜 아래에 빨간 점으로 예약 개수 표시 (최대 5개)

#### 3. ✅ 날짜별 예약 리스트
- **테스트**: 캘린더에서 날짜 클릭
- **결과**: 
  - 하단에 해당 날짜의 예약 목록 표시
  - 시간순 정렬
  - 고객명, 서비스, 스텝, 메모 표시
  - 예약 개수 표시 (예: "6월 7일 예약 (3건)")

#### 4. ✅ 타임라인 이동
- **테스트**: 선택된 날짜를 다시 클릭
- **결과**: 
  - 미니 달력이 닫힘
  - 해당 날짜의 타임라인 페이지로 이동

#### 5. ✅ 월 이동
- **테스트**: ‹ › 버튼 클릭
- **결과**: 이전/다음 달로 이동, 예약 점 표시 유지

#### 6. ✅ 다른 날짜 선택
- **테스트**: 다른 날짜 클릭
- **결과**: 
  - 선택 강조 변경 (검은색 배경)
  - 하단 리스트가 새 날짜의 예약으로 업데이트
