# 누누아누 예약 앱 빠른 사용 가이드

## 🚀 빠른 예약 생성 (신규 기능!)

### 방법 1: 자동완성으로 빠른 예약 (추천!)
```
1. 스케줄 탭 → 상단 날짜 클릭 (미니 달력 열기)
2. 원하는 날짜 클릭
3. 검색창에 고객명 입력 (예: "김")
4. 자동완성 목록에서 고객 선택 (클릭 또는 ↓ + Enter)
5. 시술 유형 선택 → 시간 선택 → 스텝 배정 → 완료!
```
⏱️ **예상 소요 시간**: 약 30초

### 방법 2: 기존 방식
```
1. 스케줄 탭 → + 버튼 클릭
2. 날짜 선택
3. 고객 정보 입력
4. 시술 유형 선택
5. 시간 선택
6. 스텝 배정
7. 완료
```
⏱️ **예상 소요 시간**: 약 1분

## 🔍 고객 예약 이력 검색

### 이름으로 검색
```
1. 미니 달력 열기
2. 검색창에 "김철수" 입력
3. 🔍 버튼 클릭 (또는 Enter)
4. 노란색으로 표시된 날짜 클릭
5. 하단 리스트에서 예약 상세 확인
```

### 연락처로 검색
```
1. 검색창에 "010-1234" 입력
2. 🔍 버튼 클릭
3. 해당 번호의 모든 예약 표시
```

## ⌨️ 키보드 단축키

### 자동완성 네비게이션
- **↑ / ↓**: 목록에서 이동
- **Enter**: 선택
- **ESC**: 닫기
- **Tab**: 다음 필드로 이동

### 일반 단축키
- **X 버튼**: 검색 초기화
- **날짜 더블 클릭**: 타임라인으로 이동

## 💡 팁 & 트릭

### 더 빠른 예약 생성
1. 자주 예약하는 고객은 이름 첫 글자만 입력해도 자동완성에 나타남
2. 키보드만으로 조작: Tab → 입력 → ↓ → Enter
3. 검색창은 한글/숫자 모두 지원 (연락처 하이픈 자동 무시)

### 효율적인 일정 관리
1. **오늘 버튼**: 빠르게 오늘 날짜로 이동
2. **← →**: 전날/다음날 이동
3. **미니 달력의 점**: 예약 개수 (최대 5개)

### 고객 검색 활용
1. **부분 검색 가능**: "김" → 김철수, 김영희 모두 표시
2. **연락처 뒷자리만으로도 검색 가능**: "5678" → 010-1234-5678
3. **검색 결과 안내 배너**: 총 몇 건의 예약이 몇 일에 걸쳐 있는지 표시

## 🎯 시나리오별 가이드

### 시나리오 1: 단골 고객 예약 (30초)
```
문의: "김철수님이 내일 2시에 헤어 예약하고 싶대요"

1. 날짜 클릭 → 내일 선택
2. "김철" 입력 → 김철수 클릭
3. 헤어 선택 → 14:00 선택 → 보니 스텝 선택 → 완료
```

### 시나리오 2: 신규 고객 예약 (1분)
```
문의: "새 고객 박지은님 010-9999-1234, 모레 오전 11시 메이크업"

1. + 버튼 → 모레 선택
2. "박지은" 입력 → "010-9999-1234" 입력
3. 메이크업 선택 → 11:00 선택 → 지현 스텝 선택 → 완료
```

### 시나리오 3: 고객 예약 이력 확인 (20초)
```
문의: "최민지님 이번 달에 언제 오셨었죠?"

1. 미니 달력 → "최민지" 입력 → 🔍 클릭
2. 노란색 날짜들 확인 (3일, 15일, 22일)
3. 각 날짜 클릭하여 상세 시간 확인
```

### 시나리오 4: 예약 변경 (40초)
```
문의: "김철수님 내일 2시를 3시로 변경해주세요"

1. 내일 날짜 → 김철수님 예약 클릭
2. 편집 버튼 → 종료 시간 15:00으로 변경
3. 저장
```

## 🎨 화면별 기능

### 📥 수신함 (Inbox)
- 대기 중인 예약 요청 확인
- 확정/거절 처리
- 시간 조정 및 스텝 배정

### 📅 스케줄 (Schedule)
- 타임라인 뷰 (04:00~22:00)
- 실시간 예약 현황
- 드래그로 시간 조정 (준비 중)
- **NEW**: 미니 달력 검색 & 자동완성

### 👥 고객 (Customers)
- 전체 고객 목록
- 예약 이력 확인
- 고객별 메모 관리

### 👨‍💼 스텝 (Staff)
- 스텝 목록 및 상태
- 개인 일정 확인
- 스텝별 예약 통계

## 🆘 문제 해결

### 자동완성이 나타나지 않아요
- Supabase 고객 데이터가 로드되었는지 확인
- 고객 탭에서 고객 목록 확인
- 브라우저 새로고침 (F5)

### 검색 결과가 없어요
- 철자가 정확한지 확인
- 부분 검색 시도 (예: "김철수" → "김철")
- 연락처는 숫자만 입력 (하이픈 자동 제거됨)

### 키보드로 선택이 안 돼요
- 검색창이 포커스되어 있는지 확인
- ↑↓ 키로 이동 후 Enter
- 또는 마우스 클릭 사용
