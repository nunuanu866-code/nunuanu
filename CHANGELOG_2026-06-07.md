# 변경 이력 - 2026년 6월 7일

## 🎉 주요 업데이트: 고객 검색 자동완성 기능

### 새로운 기능

#### ✨ 실시간 자동완성
- 검색창에 고객명 또는 연락처 입력 시 실시간으로 일치하는 고객 목록 표시
- 최대 5개의 고객을 우선순위에 따라 표시
- 부분 검색 지원 (예: "김" → 김철수, 김영희 등)
- 연락처 하이픈 자동 제거하여 검색

#### ⌨️ 키보드 네비게이션
- **↑↓ 화살표 키**: 자동완성 목록에서 위아래 이동
- **Enter 키**: 포커스된 고객 선택 또는 검색 실행
- **ESC 키**: 자동완성 드롭다운 닫기
- 마우스 호버 시 자동 하이라이트

#### 🚀 빠른 예약 생성
- 자동완성에서 고객 선택 시 즉시 예약 생성 플로우 시작
- 고객 정보 자동 입력 (이름, 연락처, PIN, 메모)
- 고객 선택 단계(S2) 자동 건너뛰기
- 시술 유형 선택(S3)부터 바로 시작

### 성능 최적화

#### useMemo 사용
- 검색어 변경 시에만 필터링 재계산
- 불필요한 리렌더링 방지
- 최대 5개로 제한하여 DOM 크기 최소화

#### 이벤트 최적화
- 포커스 아웃 시 200ms 지연으로 클릭 이벤트 우선 처리
- 키보드 이벤트 preventDefault로 기본 동작 차단
