# 테스트 가이드 - 자동완성 기능

## 🧪 테스트 환경 준비

### 1. 파일 열기
```
파일 경로: C:\Users\tyson\OneDrive\Documents\누누아누 고객 예약 스케줄\nununanu-app\admin.html
브라우저: Chrome, Edge, Firefox 등
```

### 2. 로그인
- 비밀번호: `nunu1122!`

### 3. 데이터 확인
- 고객 탭에서 등록된 고객 확인
- 최소 2-3명의 고객이 있어야 자동완성 테스트 가능

## ✅ 테스트 체크리스트

### Phase 1: 기본 기능 테스트

#### 1.1 미니 달력 열기
- [ ] 스케줄 탭 이동
- [ ] 상단 날짜 영역 클릭
- [ ] 미니 달력 정상 표시
- [ ] **예상 결과**: 에러 없이 미니 달력 열림

#### 1.2 검색창 표시
- [ ] 검색창이 미니 달력 상단에 표시됨
- [ ] 플레이스홀더 텍스트: "고객명 또는 연락처 검색"
- [ ] **예상 결과**: 검색창 정상 표시

### Phase 2: 자동완성 테스트

#### 2.1 고객명으로 검색
```
입력: "김"
```
- [ ] 자동완성 드롭다운 즉시 표시
- [ ] "김"으로 시작하는 고객 목록 표시 (최대 5개)
- [ ] 각 항목에 이름, 연락처 표시
- [ ] **예상 결과**: 실시간으로 목록 업데이트

#### 2.2 연락처로 검색
```
입력: "010"
```
- [ ] 010으로 시작하는 연락처 고객 표시
- [ ] 하이픈 유무 상관없이 검색됨
- [ ] **예상 결과**: 연락처 검색 정상 작동

#### 2.3 부분 검색
```
입력: "철"
```
- [ ] "철"이 포함된 이름의 고객 표시 (예: 김철수, 이철민 등)
- [ ] **예상 결과**: 부분 일치 검색 작동

### Phase 3: 키보드 네비게이션 테스트

#### 3.1 화살표 키
```
1. 검색창에 "김" 입력
2. ↓ 키 누르기
```
- [ ] 첫 번째 항목 하이라이트 (회색 배경)
- [ ] ↓ 반복 시 다음 항목으로 이동
- [ ] ↑ 키로 이전 항목 이동
- [ ] **예상 결과**: 키보드로 목록 탐색 가능

#### 3.2 Enter 키
```
1. ↓로 원하는 고객 선택
2. Enter 키
```
- [ ] 선택된 고객으로 예약 생성 시작
- [ ] 미니 달력 자동 닫힘
- [ ] 시술 유형 선택 화면 표시
- [ ] **예상 결과**: 고객 정보 자동 입력됨

#### 3.3 ESC 키
```
1. 검색창에 입력
2. ESC 키
```
- [ ] 자동완성 드롭다운 닫힘
- [ ] 검색어는 유지됨
- [ ] **예상 결과**: 드롭다운만 닫힘

### Phase 4: 마우스 인터랙션 테스트

#### 4.1 마우스 호버
```
1. 자동완성 목록 표시
2. 마우스를 각 항목에 올림
```
- [ ] 호버된 항목 배경색 변경 (#F3F4F6)
- [ ] 다른 항목으로 이동 시 하이라이트 이동
- [ ] **예상 결과**: 시각적 피드백 정상

#### 4.2 마우스 클릭
```
1. 자동완성 목록에서 고객 클릭
```
- [ ] 선택된 고객으로 예약 생성 시작
- [ ] 미니 달력 닫힘
- [ ] 고객 정보 자동 입력 확인
- [ ] **예상 결과**: 원클릭으로 고객 선택 완료

### Phase 5: 예약 생성 플로우 테스트

#### 5.1 자동완성으로 예약 생성
```
1. 날짜 클릭 (예: 내일)
2. 검색창에 "김철수" 입력
3. 김철수 선택
```
- [ ] S3 (시술 유형 선택) 화면으로 이동
- [ ] S2 (고객 선택) 단계 건너뜀
- [ ] 상단에 "김철수" 이름 표시
- [ ] **예상 결과**: 2단계 단축 확인

#### 5.2 나머지 단계 완료
```
4. 시술 유형 선택 (예: 헤어)
5. 시간 선택 (예: 14:00)
6. 스텝 선택 (예: 보니)
7. 확정
```
- [ ] 모든 단계 정상 진행
- [ ] 예약 생성 완료
- [ ] 타임라인에 예약 표시
- [ ] **예상 결과**: 예약 성공

### Phase 6: 기존 기능 호환성 테스트

#### 6.1 검색 기능 (🔍 버튼)
```
1. 검색창에 "김철수" 입력
2. 🔍 버튼 클릭
```
- [ ] 자동완성 무시됨
- [ ] 검색 모드 활성화
- [ ] 노란색 날짜 하이라이트
- [ ] 검색 결과 안내 배너 표시
- [ ] **예상 결과**: 기존 검색 기능 정상 작동

#### 6.2 X 버튼
```
1. 검색어 입력
2. X 버튼 클릭
```
- [ ] 검색어 초기화
- [ ] 자동완성 닫힘
- [ ] 검색 모드 해제
- [ ] **예상 결과**: 초기화 정상 작동

#### 6.3 S1Date 미니 달력 (예약 생성 첫 단계)
```
1. + 버튼 클릭
2. 미니 달력 표시
```
- [ ] 미니 달력 정상 표시
- [ ] 검색창 표시 (자동완성 없음)
- [ ] 날짜 선택 정상 작동
- [ ] **예상 결과**: 에러 없이 작동

### Phase 7: 엣지 케이스 테스트

#### 7.1 고객 없을 때
```
조건: supaCustomers가 빈 배열
```
- [ ] 자동완성 표시 안 됨
- [ ] 검색 기능은 정상 작동
- [ ] **예상 결과**: 에러 없음

#### 7.2 검색 결과 없을 때
```
입력: "존재하지않는이름12345"
```
- [ ] 자동완성 드롭다운 표시 안 됨
- [ ] 에러 없음
- [ ] **예상 결과**: 빈 결과 처리 정상

#### 7.3 특수문자 입력
```
입력: "김-철-수"
```
- [ ] 하이픈 자동 제거되어 검색
- [ ] 연락처 검색 시 하이픈 무시
- [ ] **예상 결과**: 특수문자 처리 정상

#### 7.4 빠른 타이핑
```
빠르게 연속 입력: "김철수"
```
- [ ] 모든 문자 입력 반영
- [ ] 자동완성 정상 업데이트
- [ ] 성능 문제 없음
- [ ] **예상 결과**: 반응성 정상

## 🐛 문제 발생 시 체크리스트

### 자동완성이 표시되지 않을 때
- [ ] 고객 탭에서 고객이 등록되어 있는지 확인
- [ ] 브라우저 콘솔 에러 확인 (F12)
- [ ] 페이지 새로고침 (F5)

### 예약 생성이 안 될 때
- [ ] 모든 필수 정보 입력 확인
- [ ] 스텝이 활성화되어 있는지 확인
- [ ] 시간 중복 확인

### 미니 달력이 안 열릴 때
- [ ] 브라우저 콘솔 확인
- [ ] admin.html 파일 재로드
- [ ] 캐시 삭제 후 재시도
